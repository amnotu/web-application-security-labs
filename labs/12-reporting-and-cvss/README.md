# Lab 12: Reporting and CVSS

## Objective

Convert selected lab findings into a public-safe report summary with severity, CVSS-style scoring, evidence handling, and remediation.

## Methodology Phase

PTES Phase: Reporting

This lab focuses on communicating findings clearly without exposing sensitive artifacts.

## Scope

This report summarizes selected findings from the local lab portfolio.

Included evidence comes from:

- Lab 07: Injection Attacks
- Lab 08: Cross-Site Scripting
- Lab 09: Identification and Authentication Failures
- Lab 10: Broken Access Control
- Lab 11: Miscellaneous Web Vulnerabilities

## Deliverables

| File | Purpose |
|---|---|
| `cvss-worksheet.md` | CVSS-style severity worksheet for selected findings. |
| `public-report-summary.md` | Public-safe summary report for portfolio use. |
| `01-public-report-summary.png` | Screenshot of the public report summary. |

## Reporting Notes

Scores are based on the local lab evidence and are not production ratings.

The report avoids raw exploit output, cookies, session tokens, credentials, packet captures, full decrypted values, and course-provided binaries.
